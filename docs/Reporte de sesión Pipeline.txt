  SyntaxChecker → JsonToGraph → FunctionChecker → CSPEval → GecodeInfo

  casos. Fix en ParseNodeBlock de UCSPJson.pas.

  4. Conclusión sobre GecodeInfo

  La herramienta funciona como oráculo de incertidumbre acotada: dado un modelo parcialmente resuelto, cuantifica exactamente cuánta información gana fijar cada variable a cada valor.
  Esto permite tomar decisiones de observación óptimas — la variable que, al fijarse, más reduce el espacio es la de mayor ganancia de información.
